Summary

This report examines the relationship between mileage (mpg or miles per gallon) and transmission (am) type of the car. We have to check whether;

  • an automatic or manual transmission is better for MPG and,
  • quantify the MPG difference between automatic and manual transmissions

Based on the analysis, I have concluded that,

  1. manual transmission on average performs 7.25 miles per gallon better than automatic transmission. However, transmission type (am) alone explains only 33.85% of the variance.

  2. After adding other variables, manual transmission alone improves mgp by 2.94 compared to automatic transmission while holding weight and qsec (1/4 mile time) constant. These three variables (am, wt, qsec) collectively explain 83.36% variance of the outcome.

About the assignmenet

You work for Motor Trend, a magazine about the automobile industry. Looking at a data set of a collection of cars, they are interested in exploring the relationship between a set of variables and miles per gallon (MPG) (outcome).

The data was extracted from the 1974 Motor Trend US magazine, and comprises fuel consumption and 10 aspects of automobile design and performance for 32 automobiles (1973–74 models). The data frame has 32 observations on 11 (numeric) variables. the variable description is provided below.

  • mpg Miles/(US) gallon
  • cy Number of cylinders
  • disp Displacement (cu.in.)
    hp Gross horsepower
    drat Rear axle ratio
    wt Weight (1000 lbs)
    qsec 1/4 mile time
    vs Engine (0 = V-shaped, 1 = straight)
    am Transmission (0 = automatic, 1 = manual)
    gear Number of forward gears
    carb Number of carburetors

Statistical Inference

To check whether cars mileage (mpg) is impacted by transmission type (am), we will set a null and alternative hypothesis; – null hypthesis: mpg of manual and automatic cars are no different. – alternative hypothesis: there is significant difference between mpg manual vs automatic cars.

We use t-test to verify this.

test <- t.test(mpg ~ am, data = mtcars)

test$p.value
## [1] 0.001373638

Based on p.value of r, test$p.value, which is significantly less than 0.05, we reject null hyothesis. There is significant difference in mpg of two types of cars.

We can also check for average mileage (mpg) for automatic and manual cars.

autoMeanMPG <- mean(mtcars$mpg[mtcars$am == 0])

manualMeanMPG <- mean(mtcars$mpg[mtcars$am == 1])
